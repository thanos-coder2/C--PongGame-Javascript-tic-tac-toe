- 🧱 HTML DOM Structure
- 🎨 CSS Styling & Animations
- ⚙️ JavaScript Event Handling

# 📌 Description

This application is a classic **Tic Tac Toe** game designed for two players.
Players take turns as **X** and **O** until one player wins or the game ends in a draw.

This project was created as a university assignment with the goal of practicing:

* 🧱 HTML DOM Structure
* 🎨 CSS Styling & Animations
* ⚙️ JavaScript Event Handling
* 🧠 Game Logic

---

<div align="center">

| 🔄 Restart Button    | Restart the game at any time         |
| ✅ Win Detection      | Detects the winning player           |
